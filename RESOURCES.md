# TypeScript Learning Resources

## Knowledge

- [TypeScript Handbook — Array Methods](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html)
  Covers find, filter, map, some, every, reduce. Use for: teaching standard library methods.
- [MDN: Array.prototype.filter](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter)
  Clear examples and explanations. Use for: filtering exercises.
- [MDN: Array.prototype.reduce](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/reduce)
  Comprehensive guide to reduction. Use for: aggregation exercises.
- [TypeScript Deep Dive: Type Narrowing](https://basarat.gitbook.io/typescript/type-system/type-narrowing)
  Covers truthy/falsy checks, type guards, control flow analysis. Use for: undefined/null handling.

## Wisdom (Communities)

- [r/typescript](https://reddit.com/r/typescript)
  Active community, good for real-world questions and pattern discussions.
- [TypeScript Discord](https://discord.gg/typescript)
  Real-time help, code review, pattern discussions.
