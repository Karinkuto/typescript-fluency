# Practice Exercises — Building Fluency

These exercises are designed to build reps with decision-making, not just syntax.
For each problem:
1. Describe what you're trying to do in plain English FIRST
2. Decide which method(s) to use and WHY
3. Write the code
4. Run it and verify

## Exercise Sets

### Set A: Choose the Right Method
For each scenario, pick the single best method and explain why.

### Set B: Combine Methods
Chain 2-3 methods to solve a problem in one pipeline.

### Set C: Real-World Patterns
Apply methods to realistic data structures.

### Set D: Boolean Logic Drills
Explicit practice on conditions — the known weak spot.
