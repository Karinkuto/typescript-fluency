# Searching and Finding

Student can implement linear search using for...of with a conditional, and understands Array.find() as a shorthand. Understands that searching means finding a matching entry based on a condition.

## Evidence
- Implemented `for (...) { if (person.name === "Amen") }` correctly
- Later adopted `find(...)` after seeing it
- Correctly described the algorithm in plain English before coding it

## Implications
- Can teach filtering (find returns first match; filter returns all matches) as a natural next step
- Aggregation exercises that build on search+update pattern are appropriate
